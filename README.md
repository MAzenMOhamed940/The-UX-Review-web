🎨 The UX Review — Blog Landing Page
A responsive blog landing page built using HTML5 and CSS3, inspired by the Neubrutalism UI design style.
This project focuses on modern layout techniques using Flexbox, responsive design, and clean UI structure.

🚀 Live Demo
🔗 https://lnkd.in/d7jSFCyT

📌 Features
✅ Fully responsive design across all screen sizes
✅ Fixed navigation bar with responsive behavior
✅ Bold Neubrutalism-style UI (strong colors & sharp layouts)
✅ Smooth hover effects and UI interactions

🛠️ Tech Stack
HTML5 — page structure
CSS3 — styling and layout
Flexbox — responsive layout system
Media Queries — responsive design
Box Model & box-sizing — layout control
Pseudo-classes & pseudo-elements — UI enhancements
Transitions & Transforms — animations and interactions

🎯 What I Learned
Building responsive layouts using Flexbox
Writing clean and maintainable CSS
Using media queries for different screen sizes
Controlling layout using box-sizing and the box model
Enhancing UI with hover effects and transitions
Translating modern UI designs into real code
